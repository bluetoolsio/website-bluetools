# BlueTools brand assets

Canonical BlueTools logo and icon exports.

## Files

- `bluetools-logo.svg` — full BlueTools logo, vector
- `bluetools-logo.png` — full BlueTools logo, transparent PNG
- `bluetools-icon.svg` — BlueTools mark/icon, vector
- `bluetools-icon.png` — BlueTools mark/icon, PNG
- `bluetools-logo-source.png` — source logo export supplied for the repo
- `bluetools-icon-source.png` — source icon export supplied for the repo
- `bluetools-logo-source.psd` — editable Photoshop source for the logo

The website favicon uses the BlueTools icon.
